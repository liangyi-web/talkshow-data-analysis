# talkshow-data-analysis

## Project Description
This project conducts data analysis on a talk show subtitle and danmu dataset using Python.

## Repository Contents
- `Python code with detailed notation.ipynb`  
  The original Python notebook containing all analysis code.

- `data samples the first 10 rows of datasets used for analysis.csv`  
  A public data sample consisting of the first 10 rows of the dataset used in the analysis.

## Data Availability
Due to data size and privacy considerations, only the first 10 rows of the dataset are uploaded for public access.
